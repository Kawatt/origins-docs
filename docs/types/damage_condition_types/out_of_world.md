#   Out of World

[Damage Condition Type](../damage_condition_types.md)

Checks whether the damage source is out of world.

Type ID: `origins:out_of_world`

!!! danger

    This damage condition type has been **deprecated** and may be removed in a future version. Please use [In Tag (Damage Condition Type)](in_tag.md) instead.

### Fields

_None._


### Examples

```json
"damage_condition": {
    "type": "origins:out_of_world"
}
```
